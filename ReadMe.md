## Project Overview

This website is a simple, responsive informational site built as the first part of a web development project. It provides an introduction to the organization, highlights services and events, and helps visitors navigate key pages.

## Website Goals and Objectives

- Present a clean and easy-to-read site structure for visitors.
- Communicate the purpose of the organization and available resources.
- Offer clear navigation between pages such as Home, About, Contact, Policies, and Events.
- Support future content growth and improvements.

## Key Features and Functionality

- Multi-page HTML site using semantic structure.
- Navigation links for fast access to key pages.
- Organized content sections with headings, text, and imagery.
- A dedicated contact page and privacy/policy page.
- Event details page for past or upcoming activities.

## Part 1 Details

Part 1 focuses on building the foundational website structure and content:

- Create the main page (`index.html`) with an overview and navigation.
- Add an About page (`About.html`) describing the organization.
- Add a Contact page (`contact.html`) for getting in touch.
- Add a Policies page (`policies.html`) for terms and privacy information.
- Add an Events page (`Previousevents.html`) to show past or planned events.
- Include images in the `Images/` folder to support page content.
